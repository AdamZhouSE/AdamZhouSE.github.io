C++字符串,向量和数组

二、字符串，向量和数组

作用域操作符::

含义：编译器应从操作符左侧的作用域中寻找右边那个名字

比如std::cin是要使用命名空间std中的cin

这样做是为了避免某些名字被重复使用

使用using声明可以更加简便的使用命名空间中的成员。

1. String类型

表示可变长的字符串

1.1 定义和初始化

  1. 使用需要头文件和命名空间

    #include<string>
using std::string;

  2. 初始化

    string s1; // 空字符串
string s2 = s1; // s2是s1的副本
string s3 = "abc";
string s4(10, 'c'); // 十个c
1.2 string对象上的操作

  1. 使用getline读取一整行

    string line;
getline(cin, line);

    当读取到换行符时结束，并将读到的内容存入string中，但是不存换行符

  2. empty和size操作

    str.empty()根据字符串是否为空返回一个布尔值

    str.size()返回字符串长度

    使用visual studio编程的时候，碰到过一个warning，将无符号数赋值给了有符号数。

    这是由于size函数的返回值类型引起的，它是一个string::size_type类型的值，是无符号类型的，

    可以用auto来推断变量的值

    auto len = str.size();

  3. 比较，赋值，相加

    string s1 = "abc";
string s2 = "xcvd";
// s1 < s2 比较第一个不同的字符
string s3 = s1; // 赋值
string s4 = s1 + s2; // abcxcvd

  4. 处理string中的字符

    #include<cctype>
isalnum(c);
isalpha(c);
islower(c);
isupper(c);
ispunct(c); // 是标点符号为真

2. Vector类型

可变长数组，类似于Java中的ArrayList

2.1 初始化
vector<T> v;
vector<T> v1 = {a, b, c}; // 注意是花括号
vector<T> v2{a, b};     // 花括号，两个元素
vector<int> v3(10, 1); // 圆括号 10个元素，每个初始化为1
2.2 操作
vector<int> v;
v.push_back(1); // 添加1
v.pop_back(); // 去掉最后一个数据
v.empty();
v.size();
v[n]; // 注意下标只能访问，但是不能够添加元素

3. 迭代器

访问元素的作用

string s = "abcde";
if (s.begin() != s.end()) { // 确保s非空
  
}

4. 数组

4.1 指针和数组
int ia[] = {1, 2, 3, 4, 5};
auto p(ia); // 整型指针，指向ia的第一个元素
p++; // 指向第一个元素
